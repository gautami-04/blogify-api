# Blogify API

A minimal backend foundation for the Blogify project (Module 1 integration checkpoint).

## Getting started

Install dependencies:

```bash
npm install
```

Run in development (uses nodemon):

```bash
npm run dev
```

Open http://localhost:3000 to see "Blogify API is running!".

## Project structure

- `src/` - application source code
  - `controllers/`
  - `routes/`
  - `services/`
  - `utils/`
  - `index.js` - main server entry
